Detailed Description

Clasa CalculeazaVarsta se va folosi pentru a calcula varsta in functie de anul nasterii introdus de utilizator si de anul curent. Este nevoie de aceasta clasa pentru a sti daca persoana care se logheaza are intre 16 si 100 de ani sau daca a introdus o data valida. /summary>
